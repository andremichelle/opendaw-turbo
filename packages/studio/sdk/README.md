# openDAW SDK

## 1. Add everything in one shot

`npm install @opendaw/studio-sdk`

## 2. You now have access to the following packages

* `@opendaw/lib-std`
* `@opendaw/lib-box`
* `@opendaw/lib-dsp`
* `@opendaw/lib-runtime`
* `@opendaw/lib-fusion`
* `@opendaw/studio-enums`
* `@opendaw/studio-boxes`
* `@opendaw/studio-adapters`
* `@opendaw/studio-core`
